# Ecommerce App UI

A modern, responsive ecommerce frontend built with **HTML**, **Tailwind CSS**, and **JavaScript**.

## Features

- Product listing grid
- Product detail modal
- Add to cart functionality
- Responsive design for mobile and desktop
- Minimal, clean UI

## Technologies

- **HTML5**: Semantic markup
- **Tailwind CSS**: Utility-first styling
- **JavaScript**: UI interactions and cart logic

## Getting Started

1. **Clone the repository**

   ```bash
   git clone https://github.com/your-username/ecommerce-app-ui.git
   cd ecommerce-app-ui
   ```

2. **Open `index.html` in your browser**

## Folder Structure

```
├── index.html
├── assets/
│   └── images/
├── css/
│   └── tailwind.css
├── js/
│   └── main.js
```

## Customization

- Update product data in `main.js`
- Modify styles in `tailwind.css`
- Add images to `assets/images/`
